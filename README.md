# 📊 ABHAY-SQL: Zero to Advanced SQL Learning

Welcome to the **ABHAY-SQL** repository! This project serves as a comprehensive collection of structured query language (SQL) scripts designed to document my journey from foundational database concepts to advanced data manipulation techniques. 

The scripts housed here focus on database schema design, data integrity, optimized queries, and complex analytical functions tailored for data science workflows.

---

## 📁 Repository Structure

The repository is organized by core SQL concepts, progressing from basic structural setup to advanced analytical queries:

```text
├── .vscode/                     # Editor configurations
├── setup_database.sql           # Initial database setup and configuration
├── createusertable.sql          # Table creation scripts and schema definition
├── primarykey.sql               # Implementation of Primary Key constraints
├── Foreignkey.sql               # Implementation of Referential Integrity (Foreign Keys)
├── AbhaySQL.sql                 # General query exercises and core syntax
├── JIons to all Data science imp.sql  # Essential Joins for Data Science workflows
└── Window funtion and CTEs.sql  # Advanced Window Functions and Common Table Expressions
